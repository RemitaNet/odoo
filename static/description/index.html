Remita Payment Gateway

Seamless payment processing with Remita

Features

  • Easy integration with Remita payment services
  • Support for both test and live environments
  • Secure payment processing
  • Full transaction history and reporting
  • Multiple payment methods supported

How to Configure

  1. Install the module
  2. Go to Invoicing > Configuration > Payment Acquirers
  3. Configure the Remita payment acquirer with your credentials
  4. Set the environment to Test or Live as needed
